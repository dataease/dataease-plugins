### 步骤一：登录 DataEase 系统

打开浏览器，访问 DataEase 系统并完成登录。

### 步骤二：选择 Trino 数据源

在数据源管理页面，找到并点击 **Trino** 图标，进入 Trino 数据源配置页面。

### 步骤三：填写连接信息

在配置表单中填入以下信息：

| 参数               | 说明                                                                   |
| ------------------ | ---------------------------------------------------------------------- |
| 主机名/IP地址      | Trino 服务器的主机名或 IP 地址                                         |
| 端口               | Trino 服务监听的端口号，默认 8080                                      |
| Catalog            | Trino 中的目录名称，对应底层数据源连接器（如 `mysql`、`hive`、`postgresql` 等） |
| Schema             | Catalog 下的模式名称，对应底层数据源中的数据库或命名空间               |
| 用户名             | 具有访问权限的 Trino 账号                                              |
| 密码               | 对应账号的登录密码                                                     |
| 额外的 JDBC 连接字符串 | 可选，用于追加自定义 JDBC 连接参数                                 |

填写完成后，点击**校验连接**按钮。校验通过即表示成功接入 Trino 数据源。

## 注意事项

- 请确保网络环境可以正常访问 Trino 服务器。
- Catalog 与 Schema 必须为 Trino 中已配置且当前账号有访问权限的有效值。
- 所填写的账号需具备对应 Catalog 和 Schema 的查询权限。
